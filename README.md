# QuickQueue-WotLK (3.3.5)

Un addon tout-en-un pour simplifier l'expérience des donjons sur **WoW WotLK 3.3.5**. Il automatise les interactions sociales et les tags en donjon via un bouton de contrôle sur la minimap.

## ✨ Fonctionnalités
* **🟢 Bouton Minimap :** Un indicateur visuel (Vert = ON, Rouge = OFF) pour activer ou désactiver l'addon en un clic. Déplaçable par clic-droit.
* **⚔️ Auto-Queue :** Propose automatiquement de s'inscrire en Recherche de Groupe (LFD) après 5 secondes de connexion ou de sortie d'instance.
* **👋 Politesse Automatique :** Envoie un message de bienvenue ("Hello everyone...") 10 secondes après être entré dans un donjon.
* **🛡️ Marquage du Tank :** Pose automatiquement une icône (Etoile) sur le Tank du groupe après 12 secondes d'instance.
* **💰 Auto-Loot Roll :** Confirme automatiquement les jets de dés si vous choisissez "Cupidité" (Greed) pour éviter les fenêtres pop-up.

## 🛠️ Installation
1. Téléchargez le ZIP.
2. Dézippez-le.
3. Placez le dossier `QuickQueue-WotLK` dans votre dossier `Interface/AddOns/`.
4. En jeu, cliquez sur le bouton **QQ** sur votre minimap pour gérer l'activation.

---
*Note : Pour le marquage automatique du Tank, vous devez être chef de groupe ou assistant.*
